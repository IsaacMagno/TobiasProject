# TobiasProject
## 🛠️ Projeto pessoal em desenvolvimento

#### 🖖 O projeto Tobias é um App criado por mim para ter dados e métricas sobre Autodesenvolvimento, em um estilo "RPG".

#### 🚧  Principais Tecnologias utlizadas no desenvolvimento
- NextJS
- Redux
- NodeJS
- MySQL
- TailwindCSS
